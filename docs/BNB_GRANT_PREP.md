# BNB Chain Grant Prep

## Positioning

RugBuster Syndicate AI is a multichain threat intelligence layer. The system
becomes stronger as it observes token deployments, funding patterns, holder
concentration, liquidity behavior, and rug velocity across multiple networks.

BNB Smart Chain is the next target network after Solana and BNB Chain.

## Grant Track Fit

Primary category:

- Developer Tools & Infrastructure

Secondary category:

- AI
- Security tooling
- Ecosystem analytics

## BNB-Specific Deliverables

- BNB Smart Chain collector for new BEP-20 tokens and DEX pairs.
- PancakeSwap, Biswap, and ApeSwap pair monitoring.
- Public BNB scan API mode.
- BNB Telegram alert channel.
- Recent BNB scan feed on the website.
- BNB on-chain registry for module-level scan proofs.
- Builder documentation for integrating score and registry data.

## Evidence To Collect Before Applying

- 24-48 hours of stable BNB worker uptime.
- 300-1000 BSC token scan records.
- Telegram alert screenshots.
- Recent scan feed screenshots.
- Registry transaction hashes after BNB registry deployment.
- README section showing BNB builder usage.
- API examples using `chain=bsc`.

## Problem Statement

AI lowers the cost of automated contract exploitation and scam deployment.
Builders need real-time, public, AI-assisted threat intelligence that evaluates
new tokens and contracts as they appear, before users interact with them.

## Product Statement

RugBuster Syndicate AI monitors new BNB Smart Chain token activity, runs
multi-module risk analysis, generates concise AI-assisted verdicts, sends
real-time alerts, and stores module-level findings in an on-chain registry for
public verification.

## Not Ready Until

- BNB collector runs without affecting the BNB Chain worker.
- BNB token names and symbols appear correctly in alerts.
- The website can show recent BNB scans.
- BNB registry writes are live or a clear milestone is defined.
